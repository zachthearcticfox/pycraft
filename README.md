# pycraft
Pycraft is a work-in-progress game a bit like Minecraft but 2D.
The game is made with pygame 2.6.1
Have fun building :3

## controls
wasd | movement

g | give item

right alt | equip block from text input

1-3 | equip block

\- | give 1 stone

= | give 1 grass

left control | save world

left alt | load world

0 | enable mining mode

c | toggle cave mode
